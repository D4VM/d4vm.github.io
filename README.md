# Davit Miruashvili - Portfolio & Resume

Personal website and resume hosted on GitHub Pages.

## Built With
- HTML5
- Tailwind CSS (via CDN)
- FontAwesome
- Inter Font

## Design
The design is based on the professional PDF resume layout, featuring:
- Responsive grid structure
- System Administrator branding
- Dual language support (KA/EN)
- Print-optimized styles

## Structure
- `index.html`: Main landing page with language support
- `assets/`: Images and favicon
